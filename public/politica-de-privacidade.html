Política de Privacidade

Última atualização: Dezembro de 2024

1. Coleta de Dados

O Dracker Adapta prioriza sua privacidade. Nós não coletamos dados pessoais identificáveis em nossos servidores. Todo o processamento de texto e geração de atividades ocorre localmente no seu navegador ou através da API do Google Gemini, conforme sua configuração.

2. Armazenamento Local (LocalStorage)

Para melhorar sua experiência, armazenamos alguns dados diretamente no seu dispositivo (navegador) utilizando a tecnologia LocalStorage. Isso inclui:

  • Sua chave de API do Gemini (salva localmente para sua conveniência).
  • As atividades e abas geradas recentemente.
  • Dados da "Expedição Drácker" (turmas e membros).

Esses dados nunca saem do seu dispositivo, exceto quando você explicitamente utiliza a função de Backup/Exportar para baixar um arquivo JSON.

3. Uso de Inteligência Artificial

Utilizamos a API do Google Gemini para gerar conteúdo. Ao utilizar o sistema, os textos dos tópicos e comandos que você insere são enviados para a API do Google para processamento. Recomendamos não inserir dados sensíveis ou pessoais nos campos de geração de atividade.

4. Contato

Se tiver dúvidas sobre esta política, entre em contato com a equipe de desenvolvimento responsável pela implementação local em sua instituição.
